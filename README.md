This website was a project for Interaction Design (GDES2342), which prompted me to redesign a website for an already existing business. I chose Bordertown Coffee, a closeby and beloved fraternity house turned cafe. 

Key changes I made from the current website were clearer wording, eye-catching layout choices and visuals, and a brand redesign that changed the logo and color palette for Bordertown Coffee. Inspired by the cafe's popularity as a study spot, iconic and unique look, and its strong campus history, I styled the site to have a vintage, bookish aesthetic.
